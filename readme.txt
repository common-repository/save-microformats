== Description ==

A plugin to save posted Microformats (hCard and hCalendar) using technorati feeds.

You only have to put '&lt;!--save-hcard--&gt;' or '&lt;!--save-hcal--&gt;' where you want the button to be displayed.

== Installation ==

* Upload the `save-microformats` folder to the `/wp-content/plugins/` directory
* Activate the plugin through the 'Plugins' menu in WordPress

Thats it
